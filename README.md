# Cardboard

Specs: 

1. Client clicks on RPI 
1. Mouseclick coordinates sent through websocket
1. Click on actual device based on relative coordinate position
1. Screenshot of result rendered back to client (RPI)

Set up

1. `python server.py`
1. `open home.html`
